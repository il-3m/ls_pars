# Современный парсер лекарств с ЕИС

## Описание

Программа объединяет лучшие функции из двух версий парсеров:
- **Поиск страниц** из `ЛС-парсер-лайт.py` (поиск по МНН, даты, фильтры)
- **Парсинг страниц** с улучшенной логикой раскрытия блоков
- **Современный минималистичный интерфейс** PyQt5

## Возможности

### Поиск
- Поисковый запрос по МНН (с автодополнением из базы ЕСКЛП)
- Выбор диапазона дат (с ... по)
- Ограничение максимального количества контрактов
- Фильтр по Москве и МО
- Фильтр только по Росунимеду

### Парсинг
- Автоматическое раскрытие всех блоков на странице
- Извлечение полной информации о лекарствах:
  - Наименование
  - Страна происхождения
  - КТРУ/ОКПД2
  - Тип объекта закупки
  - Количество
  - Цена и сумма
  - Заказчик
  - Номер контракта и реестра
  - Лекарственная форма
  - Дозировка
  - Дата контракта
  - Торговое наименование
  - Номер РУ
  - **Ссылка на страницу контракта**

### Интерфейс
- Современный минималистичный дизайн
- Улучшенная читаемость и удобство использования
- Прогресс-бар выполнения
- Логирование процесса в отдельном окне
- Фильтрация результатов
- Экспорт в Excel с активными гиперссылками
- Быстрый переход к странице контракта по двойному клику

## Запуск

```bash
python "ЛС-парсер-модерн.py"
```

Или используйте bat-файл:

```bash
run_parser_gui.bat
```
