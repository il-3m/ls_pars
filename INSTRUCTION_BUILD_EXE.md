# Инструкция по созданию EXE файла парсера

## Что вам понадобится

1. **Компьютер с Windows 10/11**
2. **Установленный Python 3.8-3.12** (только для сборки, на целевом компьютере Python не нужен)
   - Скачать: https://www.python.org/downloads/
   - При установке обязательно отметьте галочку "Add Python to PATH"

## Пошаговая инструкция

### Шаг 1: Подготовка файлов

Скопируйте следующие файлы в одну папку на вашем компьютере:
- `unified_parser.py` (основной файл парсера)
- `eis_parser.py` (модуль парсинга)
- `link_finder.py` (модуль поиска ссылок)
- `requirements.txt` (файл зависимостей)
- `build_exe.bat` (скрипт сборки)

### Шаг 2: Запуск сборки

1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку с файлами:
   ```
   cd C:\путь\к\папке\с\парсером
   ```
3. Запустите скрипт сборки:
   ```
   build_exe.bat
   ```

### Шаг 3: Ожидание завершения

Процесс сборки займет **5-15 минут** в зависимости от скорости компьютера.
Скрипт автоматически:
- Установит все необходимые библиотеки
- Установит браузеры Playwright (Chromium)
- Очистит старые файлы сборки
- Создаст EXE файл с помощью PyInstaller
- Скопирует готовый файл в папку `export`

### Шаг 4: Готовый файл

После успешной сборки вы получите:
- **`export\Unified_Parser.exe`** - готовый исполняемый файл

Этот файл можно:
- ✅ Запускать на любом компьютере с Windows 10/11
- ✅ Пересылать коллегам
- ✅ Запускать без установленного Python

## Важные требования на целевом компьютере

Для работы парсера необходимо:

1. **Google Chrome** - браузер должен быть установлен
   - Парсер использует Chrome для автоматизации (Selenium)
   - Скачать: https://www.google.com/chrome/

2. **Доступ к интернету** - для работы с сайтом zakupki.gov.ru

3. **Windows 10/11** - другие версии Windows не тестировались

## Возможные проблемы и решения

### Проблема: "ОШИБКА: Python не найден в системе!"
**Решение:** Установите Python и добавьте его в PATH при установке.

### Проблема: "Не удалось установить зависимости"
**Решение:** 
- Проверьте подключение к интернету
- Попробуйте установить вручную: `pip install -r requirements.txt`

### Проблема: "Не удалось установить браузеры Playwright"
**Решение:** 
- После сборки EXE, вручную выполните команду: `python -m playwright install chromium`
- Или установите браузеры заранее перед сборкой

### Проблема: Антивирус блокирует EXE файл
**Решение:** 
- Это ложное срабатывание (часто бывает с PyInstaller)
- Добавьте файл в исключения антивируса
- Или соберите на другом компьютере

### Проблема: Парсер не запускается на другом компьютере
**Решение:**
- Убедитесь, что установлен Google Chrome
- Проверьте права администратора (если нужно)
- Попробуйте запустить от имени администратора

### Проблема: Большой размер EXE файла (~150-250 МБ)
**Это нормально!** В файл встроены все библиотеки Python, включая PyQt5, pandas, selenium, playwright.

## Дополнительные опции сборки

Если хотите изменить имя файла или параметры, отредактируйте `build_exe.bat`:

```bat
--name "Ваше_Имя.exe"  # Изменить имя файла
--icon=myicon.ico      # Добавить иконку (нужен .ico файл)
```

## Для продвинутых пользователей

Ручная сборка через командную строку:

```bat
pip install pyinstaller playwright
python -m playwright install chromium
pyinstaller --onefile --windowed --name "Unified_Parser" ^
    --hidden-import=eis_parser ^
    --hidden-import=link_finder ^
    unified_parser.py
```

Готовый файл будет в папке `dist\`

---

**Примечание:** Первый запуск EXE файла может занять несколько секунд, так как происходит распаковка встроенных библиотек.
