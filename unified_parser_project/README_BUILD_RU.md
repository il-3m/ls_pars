# 🚀 Unified Parser - Инструкция по сборке EXE

## ⚠️ КРИТИЧЕСКИ ВАЖНО: Версия Python

**НЕ ИСПОЛЬЗУЙТЕ Python 3.13!** Он слишком новый и не имеет готовых пакетов для pandas/numpy.

### ✅ Рекомендуемые версии:
- **Python 3.11** (лучшая совместимость)
- **Python 3.12** (хорошая совместимость)

### ❌ Не работает с:
- Python 3.13 и выше

---

## 📋 Шаг 1: Установка правильной версии Python

1. Удалите текущий Python 3.13 (если установлен):
   - Панель управления → Программы и компоненты → Python 3.13 → Удалить

2. Скачайте и установите Python 3.11 или 3.12:
   - Перейдите на https://www.python.org/downloads/
   - Найдите версию **3.11.x** или **3.12.x**
   - **ВАЖНО:** При установке поставьте галочку ✅ **"Add Python to PATH"**

3. Проверьте установку:
   ```cmd
   python --version
   ```
   Должно показать: `Python 3.11.x` или `Python 3.12.x`

---

## 📦 Шаг 2: Подготовка файлов

Убедитесь, что в одной папке находятся:
- `unified_parser.py`
- `eis_parser.py`
- `link_finder.py`
- `requirements.txt`
- `build_exe.bat`
- `unified_parser.spec`

---

## 🔨 Шаг 3: Запуск сборки

1. Дважды кликните на `build_exe.bat`
2. Дождитесь завершения процесса (5-15 минут)
3. Готовый файл появится в папке `export\Unified_Parser.exe`

### Что делает скрипт:
1. ✅ Проверяет версию Python (блокирует 3.13)
2. ✅ Очищает старые сборки
3. ✅ Устанавливает зависимости через зеркало (быстрее в РФ)
4. ✅ Скачивает браузеры Playwright (~150 МБ)
5. ✅ Упаковывает всё в один EXE файл
6. ✅ Копирует результат в папку `export`

## 🐛 Решение проблем

### Ошибка: "Python 3.13 пока не имеет готовых пакетов"
**Решение:** Установите Python 3.11 или 3.12 (см. Шаг 1)

### Ошибка: "Failed to install dependencies"
**Причина:** Проблемы с интернетом или блокировка PyPI  
**Решение:**
- Проверьте подключение к интернету
- Отключите VPN/прокси (если есть)
- Попробуйте запустить скрипт позже

### Ошибка: "Playwright browser installation failed"
**Решение:**
```cmd
python -m playwright install chromium
```
Затем запустите `build_exe.bat` снова

### EXE файл не запускается на другом компьютере
**Проверьте:**
1. Установлен ли Google Chrome?
2. Есть ли подключение к интернету?
3. Не блокирует ли антивирус? (добавьте в исключения)

### Парсер работает медленно или выдает ошибки
**Возможные причины:**
- Сайт заблокировал IP (используйте прокси)
- Появилась капча (решайте вручную в браузере)
- Изменилась структура сайта (нужно обновить парсер)
