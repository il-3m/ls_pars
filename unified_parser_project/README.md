# README - Unified Parser EXE

## 📦 Что в папке

Эта папка содержит все необходимые файлы для сборки EXE-версии парсера:

| Файл | Описание |
|------|----------|
| `unified_parser.py` | Главная программа (интерфейс + логика) |
| `eis_parser.py` | Модуль парсинга данных |
| `link_finder.py` | Модуль поиска ссылок |
| `requirements.txt` | Список зависимостей Python |
| `build_exe.bat` | **Скрипт автоматической сборки EXE** |
| `INSTRUCTION_BUILD_EXE.md` | Подробная инструкция по сборке |

## 🚀 Быстрый старт

### Для сборки EXE (нужно сделать один раз):

1. Убедитесь, что установлен **Python 3.8 - 3.12**
   - Скачать: https://www.python.org/downloads/
   - ⚠️ При установке отметьте галочку **"Add Python to PATH"**

2. Запустите файл **`build_exe.bat`** двойным кликом

3. Дождитесь завершения (5-15 минут)

4. Готовый файл появится в папке: **`export\Unified_Parser.exe`**

### Для использования готового EXE:

1. Скопируйте `export\Unified_Parser.exe` в любую папку
2. Запустите двойным кликом
3. Работайте! ✅

## ✅ Требования на компьютере пользователя

Для работы готового EXE файла нужно:
- Windows 10 или 11
- Google Chrome (любая версия)
- Интернет
- **Python НЕ требуется!**

## 🔧 Если возникли проблемы при сборке

### Ошибка "Python not found"
Установите Python и добавьте его в PATH при установке.

### Ошибка при загрузке пакетов
Проверьте интернет-соединение или используйте зеркало:
```bat
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple --default-timeout=60
```

### Ошибка Playwright
Запустите `build_exe.bat` от имени администратора.

### Антивирус блокирует процесс
Временно отключите антивирус или добавьте папку в исключения.

## 📊 Характеристики EXE файла

- **Размер**: ~300-400 МБ
- **Включает**: 
  - Интерпретатор Python
  - Все библиотеки (PyQt5, Playwright, Selenium, pandas)
  - Браузер Chromium (~150 МБ)
- **Работает**: На любом Windows 10/11 без установки Python

## 📝 Примечания

- Первый запуск EXE может занять несколько секунд (распаковка)
- Программа создаёт лог-файл `unified_parser_log.txt` в той же папке
- Для парсинга требуется стабильный интернет
