# Инструкция по использованию парсера лекарств с ЕИС

## Быстрый старт

### Вариант 1: Через меню (рекомендуется)

1. Запустите файл **`run_parser.bat`** (дважды кликните по файлу)
2. В появившемся меню выберите нужный режим:
   - **Режим 1**: Поиск и парсинг контрактов (автоматический поиск)
   - **Режим 2**: Парсинг по прямой ссылке (если у вас уже есть URL контракта)

---

## Режим 1: Автоматический поиск и парсинг контрактов

Этот режим автоматически находит контракты на портале zakupki.gov.ru по названию лекарственного средства и парсит данные из найденных страниц.

### Как использовать:

1. **Запустите `run_parser.bat`**
2. **Введите `1`** и нажмите Enter
3. **Откроется окно программы**, где нужно заполнить:

   | Поле | Описание | Пример |
   |------|----------|--------|
   | **Поисковый запрос (МНН)** | Название лекарственного вещества | `АЗИТРОМИЦИН`, `ПАРАЦЕТАМОЛ`, `ИНСУЛИН` |
   | **Дата с** | Начальная дата поиска контрактов | По умолчанию: 3 месяца назад |
   | **Дата по** | Конечная дата поиска контрактов | По умолчанию: сегодня |
   | **Макс. контрактов** | Сколько контрактов обработать | `20` (по умолчанию) |

4. **Фильтры (опционально):**
   - ☑ **Искать только в Москве и МО** — искать контракты заказчиков из Москвы и Московской области
   - ☑ **Искать только в Росунимеде** — искать контракты Российского университета медицины
   
   ⚠️ **Внимание:** Можно выбрать только один фильтр (они исключают друг друга)

5. **Загрузить базу МНН (опционально):**
   - Нажмите кнопку, если у вас есть файл-справочник с названиями лекарств (.xlsx)
   - Это поможет с автодополнением при вводе названия

6. **Нажмите кнопку «Найти закупки»**

7. **Процесс работы:**
   - Программа автоматически найдёт контракты на zakupki.gov.ru
   - Откроет каждый контракт и извлечёт данные о лекарствах
   - Результаты появятся в таблице внизу окна
   - Прогресс отображается в реальном времени

8. **После завершения:**
   - Нажмите **«Экспорт в Excel»** для сохранения результатов
   - Файл сохранится в формате .xlsx с гиперссылками на контракты

---

## Режим 2: Парсинг по прямой ссылке

Используйте этот режим, если у вас уже есть ссылка на конкретную страницу контракта.

### Как использовать:

1. **Запустите `run_parser.bat`**
2. **Введите `2`** и нажмите Enter
3. **Вставьте ссылку** на страницу контракта в формате:
   ```
   https://zakupki.gov.ru/epz/contract/contractCard/payment-info-and-target-of-order.html?reestrNumber=XXXXXXXXXXXXXXXXXXXX
   ```
4. **Нажмите «Начать парсинг»**

## Обработка CAPTCHA

Если портал ЕИС обнаружит автоматизированный доступ, может появиться CAPTCHA:

1. Программа приостановит выполнение
2. Появится сообщение: **«Обнаружена CAPTCHA. Продолжить?»**
3. Выберите **«Да»** — у вас будет 10 секунд на ручное прохождение CAPTCHA в браузере
4. После прохождения парсинг продолжится автоматически

## Частые вопросы

### ❓ Что такое МНН?
**МНН** (Международное непатентованное наименование) — это официальное название действующего вещества лекарства. Например:
- `АЗИТРОМИЦИН` (торговые названия: Сумамед, Азитрокс и др.)
- `ПАРАЦЕТАМОЛ` (торговые названия: Панадол, Эффералган и др.)

### ❓ Почему не находятся контракты?
- Проверьте правильность написания МНН (лучше заглавными буквами)
- Расширьте диапазон дат
- Увеличьте максимальное количество контрактов
- Попробуйте без фильтров (Москва/РОСУНИМЕД)

### ❓ Где сохраняются результаты?
При экспорте вы сами выбираете место сохранения файла. По умолчанию предлагается текущая директория.

### ❓ Можно ли остановить парсинг?
Да, закройте окно программы. Результаты, собранные до этого момента, можно экспортировать.
