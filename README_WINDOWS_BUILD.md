# 🚀 Готовое решение: Windows .exe для парсера

## Что сделано:

Созданы все необходимые файлы для сборки **автономного .exe файла**, который работает на любом Windows 10/11 **без установки Python**.

---

## 📦 Файлы для сборки:

| Файл | Назначение |
|------|------------|
| `build.bat` | 👆 Запустить двойным кликом для сборки |
| `build_exe.spec` | Конфигурация PyInstaller |
| `requirements.txt` | Зависимости Python |
| `unified_parser.py` | Основной код (GUI) |
| `eis_parser.py` | Модуль парсинга |
| `link_finder.py` | Модуль поиска ссылок |
| `esklp_smnn_20260324.xlsx` | База данных МНН |
| `BUILD_WINDOWS_EXE.md` | Подробная инструкция |

---

## 🎯 Как использовать:

### Для разработчика (сборка):

1. **На компьютере с Python:**
   - Установите Python 3.11+ с галочкой "Add to PATH"
   - Скопируйте все файлы из таблицы выше
   - Запустите `build.bat` двойным кликом
   - Через 10-20 минут получите `dist/UnifiedParser.exe`

### Для конечного пользователя:

1. **На любом Windows 10/11:**
   - Скопируйте только `UnifiedParser.exe`
   - Запустите двойным кликом
   - **Никаких установок не требуется!**

---

## ⚠️ Важно:

- **Google Chrome должен быть установлен** (программа использует его для парсинга)
- Первый запуск может быть медленным (загрузка ChromeDriver)
- Антивирус может ложно срабатывать — добавьте в исключения
