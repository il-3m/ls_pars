# Инструкция по созданию Windows .exe файла

## Что вы получите:

После сборки вы получите **UnifiedParser.exe** — полноценную программу размером ~150-200 МБ, 
которая работает на любом компьютере с Windows 10/11 **без установки Python и любых других программ**.

---

## Файлы для передачи пользователю:

Скопируйте эти файлы на компьютер с Windows:

| Файл | Описание |
|------|----------|
| `build.bat` | Скрипт автоматической сборки (двойной клик) |
| `requirements.txt` | Список зависимостей |
| `build_exe.spec` | Конфигурация для PyInstaller |
| `unified_parser.py` | Основной код программы |
| `eis_parser.py` | Модуль парсинга ЕИС |
| `link_finder.py` | Модуль поиска ссылок |
| `esklp_smnn_20260324.xlsx` | База данных МНН |

## Пошаговая инструкция:

### Шаг 1: Установка Python (только для сборки!)

**Важно:** Python нужен ТОЛЬКО на компьютере, где будет происходить сборка .exe файла. 
Конечному пользователю Python не нужен!

1. Скачайте Python 3.11 с https://www.python.org/downloads/windows/
2. Запустите установщик
3. ⚠️ **ОБЯЗАТЕЛЬНО** отметьте галочку **"Add Python to PATH"** внизу окна
4. Нажмите "Install Now"

### Шаг 2: Автоматическая сборка

1. Откройте папку с файлами
2. Дважды кликните на **`build.bat`**
3. Дождитесь завершения (может занять 10-20 минут)
4. Готовый файл появится в папке `dist/UnifiedParser.exe`

### Шаг 3: Использование готового .exe

Скопируйте **`UnifiedParser.exe`** на любой компьютер с Windows 10/11 и запускайте двойным кликом.

**Никаких дополнительных установок не требуется!**

## Технические детали:

### Что внутри .exe файла:

- Интерпретатор Python 3.x
- PyQt5 — графический интерфейс
- Selenium + WebDriver — управление браузером
- Playwright — асинхронный парсинг
- Pandas + OpenPyXL — работа с Excel
- NumPy — математические операции
- Все остальные зависимости

### Размер файла:

- **~150-200 МБ** после сжатия UPX
- Без сжатия: ~300-400 МБ

### Время первого запуска:

При первом запуске программа может работать медленнее (1-2 минуты), 
так как webdriver_manager загружает ChromeDriver. При последующих запусках — мгновенно.

## Решение проблем:

### Ошибка "Python not found" при запуске build.bat

Убедитесь, что Python установлен и добавлен в PATH. Проверьте:
```
python --version
```

### Ошибка при сборке "ModuleNotFoundError"

Запустите вручную:
```
pip install -r requirements.txt
pyinstaller --clean build_exe.spec
```

### Антивирус блокирует .exe файл

Добавьте файл в исключения антивируса. Ложное срабатывание — частое явление 
для упакованных PyInstaller приложений.

### Программа не запускается на целевом компьютере

Убедитесь, что:
- Установлен Google Chrome или Chromium
- Есть права на запуск исполняемых файлов
- Достаточно свободного места на диске

## Для разработчика:

### Повторная сборка

При изменении кода просто запустите `build.bat` снова.

### Отладка без сборки

Для тестирования изменений можно запускать напрямую:
```
python unified_parser.py --gui
```

### Уменьшение размера

Для уменьшения размера .exe можно:
1. Отключить UPX в spec-файле (`upx=False`)
2. Удалить неиспользуемые модули из `hiddenimports`
3. Использовать `--onefile` вместо `--onedir`
